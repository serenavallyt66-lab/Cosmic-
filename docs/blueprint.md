# **App Name**: Valt's Lovable

## Core Features:

- AI-powered Content Generation: Users can prompt the 'Lovable' AI to generate various content or projects, leveraging its reasoning capabilities as a tool.
- Project Organization and Filtering: Ability to view and manage projects categorized by 'All projects', 'Starred', 'Created by me', and 'Shared with me'.
- Application Navigation and Search: Provides intuitive navigation to Home, Search, and Resources sections, including a quick search shortcut (Ctrl+K).
- Recent Activity Display: Shows a list of recently accessed or created projects for quick recall and access.

## Style Guidelines:

- Primary background for the sidebar is a dark, almost black hue (#1A1A1A) for a modern, high-contrast look.
- The main content area features a vibrant gradient background, transitioning from a deep blue (#4A5B8C) at the top to a bright pink (#E87A9F) at the bottom.
- Text elements throughout the application use white or a very light grey (#FFFFFF) for optimal readability against dark and gradient backgrounds.
- Branding and interactive elements, such as the app logo and selected navigation items, incorporate subtle purple accents (#8A5F9E).
- The AI prompt input field has a dark grey background (#2A2A2A) with rounded corners, complementing the overall dark theme.
- A clean, modern sans-serif typeface (e.g., Inter or similar) is used universally to ensure legibility and a sleek aesthetic.
- A prominent, centrally aligned headline ('Let's build something, Valt') in a larger font size serves as a welcoming message.
- Minimalist, stroke-based icons are used for navigation and actions, maintaining visual consistency with the clean design.
- Small, functional icons (e.g., add, keyboard, microphone, upload) are integrated within the AI prompt input field to indicate available input methods.
- The application employs a two-column layout with a fixed-width left sidebar for navigation and a fluid main content area.
- The primary AI interaction point (prompt input field) is centrally located within the main content area, making it the focal point.
- Generous use of padding and whitespace around all UI elements creates an airy, uncluttered interface, enhancing focus and readability.
- Subtle hover effects are applied to navigation links and interactive elements, providing clear visual feedback upon user interaction.
- Smooth, non-intrusive transitions are used for state changes, such as focusing on the input field or displaying new content, to enhance the user experience.
